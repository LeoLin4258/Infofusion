# RWKV InfoFusion

RWKV InfoFusion is a web app designed for aggregating and categorizing information, powered by RWKV's AI technology.


It provides a centralized platform for organizing various types of data, with a focus on chat messages sourced from WeChat and QQ.



## Features

- **Information Aggregation**: Gather information from multiple sources and consolidate them into one centralized platform.
- **Categorization**: Classify and organize information into different categories for easier management and access.
- **AI-Powered**: Utilizes advanced AI technology from RWKV to enhance data processing and analysis capabilities.
- **Chat Message Support**: Specifically designed to handle chat messages from popular platforms such as WeChat and QQ.
- **User-friendly Interface**: Intuitive interface for seamless navigation and efficient information management.

 ![Dashboard Dark Mode](./public/dark.png)

## Supported Platforms

- **WeChat**: Import and organize chat messages from Windows WeChat conversations.
- **QQ**: Gather and categorize chat messages from Windows QQ chats.


## Technologies Used

RWKV InfoFusion is built with the following technologies:

- **[Next.js](https://nextjs.org/)**: The React framework for building fast and scalable web applications.
- **[Vercel](https://vercel.com/)**: Deployment platform for hosting InfoFusion Console and ensuring seamless scalability and performance.
- **[NextUI](https://nextui.org/)**: React component library providing pre-designed UI components for building modern web interfaces.
- **[NextJS and NextUI Dashboard Template Starter](https://github.com/Siumauricio/nextui-dashboard-template?tab=readme-ov-file#nextjs-and-nextui-dashboard-template-starter)**: Starter Template

## Getting Started

To get started with InfoFusion Console, simply follow these steps:

1. Clone the repository to your local machine.
```bash
git clone https://github.com/LeoLin4258/RWKV-InfoFusion
```

2. Install the necessary dependencies.
```bash
npm install
```

3. Run the application locally

```bash
npm run dev
```
